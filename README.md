# reading-comprehension-analysis

This project analyzes reading comprehension over time using Python.

## Overview
The dataset tracks weekly reading performance by second grade students, including:
- Book word count
- Comprehension scores
- Book difficulty
- Question difficulty

## Key Findings
- Strong negative correlation (-0.92) between book length and comprehension scores
- Difficulty levels had a smaller impact compared to word count
- Performance trends fluctuated but showed overall decline as word count increased

## Tools Used
- Python
- pandas
- matplotlib
- Excel

## Purpose
This project demonstrates data analysis skills including data cleaning, visualization, grouping, and correlation analysis using real-world inspired data.
